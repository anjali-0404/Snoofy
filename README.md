# 🌐 Snoofy – Modern Aesthetic Web Application

A clean, modern, and responsive frontend built using **React + TypeScript + Tailwind CSS**, focused on delivering a smooth and visually appealing user experience.

---

## ✨ Overview

**Snoofy** is designed as a minimal, aesthetic web interface that leverages structured UI components, reusable logic, and optimized asset handling.

The project is inspired by a reference design but implemented with:

* Improved UI/UX
* Clean architecture
* Scalable component structure

---

## 🎯 Key Features

* 🎨 Clean & minimal UI design
* ⚡ Fast and optimized performance
* 🧩 Reusable component architecture
* 📱 Fully responsive layout
* 🌙 Theme support (light/dark mode)
* 🖼️ Structured image handling system
* ✨ Smooth interactions and transitions

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** Tailwind CSS
* **State/Context:** React Context API
* **Hooks:** Custom hooks
* **Build Tool:** Vite

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/anjali-0404/Snoofy.git
```

### 2. Navigate to Project

```bash
cd Snoofy
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```
